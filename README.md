# Tiro
Minimum text editor written C.

# Features
Tiro is a text editor with syntax highlighting.

# Building
Only builds on Linux and macOS AFAIK
```
$ make
```

# Running
When writing a new document.
```
$ ./tiro
```

When editing an existing document.
```
$ ./tiro [existing document names]
```

# Instruction
|  Action | Key |
| ------- | ----|
|  Save  |  ctrl-s  |
|  Quit  |  ctrl-q  |
|  Find  |  ctrl-f  |
